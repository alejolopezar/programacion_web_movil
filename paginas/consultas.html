Pregunta, ¿qué hace el siguiente código?

<!DOCTYPE html>
<html>
<body>

<h2>JavaScript Array Sort</h2>
<p>The highest number is <span id="demo"></span>.</p>

<script>
const points = [4090, 300, 1, 5, 25, 10];
document.getElementById("demo").innerHTML = myArrayMax(points);

function myArrayMax(arr) {
let len = arr.length;
let max = -Infinity;
while (len--) {
if (arr[len] > max) {
max = arr[len];
}
}
return max;
}
</script>

</body>
</html>

Lo que este código hace es recorrer un arreglo de números y encontrar el mayor, lo hace comparando cada número y si el número comparado es mayor este se convierte en el nuevo mayor. Finalmente envia el mayor numero al elemento con el id 'demo'

Colores HSL Y HSLA



HSL significa tono, saturación y luminosidad. Se basa en la rueda de colores RGB. Cada color tiene un ángulo y un valor porcentual para los valores de saturación y luminosidad.

La saturación controla qué tan saturado debe estar el color. El 0% está completamente insaturado, mientras que el 100% está completamente saturado.

En cuanto a la claridad, controla qué tan claro u oscuro es el color. 0% es negro y 100% es blanco.
